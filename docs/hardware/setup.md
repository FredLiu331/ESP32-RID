# 硬件准备

1. 使用微雪 ESP32-C5-WIFI6-KIT-N32R8，连接 USB 数据线和稳定的 USB 5 V 供电。
2. 确认板载天线和射频区域无遮挡，测试距离按 10 m 以内规划。
3. 连接后确认串口设备存在：

   ```bash
   ls -l /dev/serial/by-id/
   ```

4. 试验现场必须是经授权且边界受控的开放环境；不得在公共空域或可能影响真实航空器的区域发射。
5. 接收机、抓包设备和发生器应使用独立供电或明确记录供电状态。

本项目固定使用 Wi-Fi Beacon，不实现 NAN。Wi-Fi 逻辑并行通过信道 6 与 149 的时分驻留实现，
并非两个独立射频同时发射。
