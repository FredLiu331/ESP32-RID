# 授权试验场说明

本设备是 RID 接收设备的测试信号源，不是航空器安全设备。所有发射必须在获得授权、边界
清晰、可控的室外试验场内进行，并遵守当地无线电、航空和频谱管理规定。

试验前必须确认：

- 试验区域没有未经协调的真实航空器或其他敏感无线业务；
- 接收机、发生器和观察人员位于明确的安全边界内；
- 使用项目规定的低功率和不超过 10 m 的测试距离；
- 现场有停止供电或立即停止测试的措施；
- 测试身份不会冒用真实无人机身份；
- 记录授权人、时间、位置、固件版本和射频配置。

不得在公共区域、机场周边、受保护频段或无法确认影响范围的地点运行。本文档不构成当地
法律许可；使用者须自行取得所需授权。
